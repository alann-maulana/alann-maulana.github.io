Magenta Account & Deletion

According to the Google Play rules, we have to provide User Data Deletion Instructions URL. If you want to delete your account for the Magenta, following these instructions:

  1. Go to Magenta's Account Menu. Click "Akun" tab within main application dashboard.
  2. Open Magenta's Detail Profile Page. Click "Detail Profil" button.
  3. Within detail profile page, Click the "triple dot" icon button on top right corner. Then click "Hapus Akun Permanen" menu.
  4. There will be a popup confirmation dialog. Click "Hapus Akun" to start account deletion.

By doing these, you will delete account data and others related data like:

  1. Account
  2. Patient data
  3. Visit reports
  4. Any other log created
